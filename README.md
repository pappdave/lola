# Lola
Lola is a dog who starts at (0.5, 0.5) and always eats the closest treat to her.
An input file contains the coordinates of the treats, the output is the length of her path.

# Directory structure
## bin
##### lola_bruteforce.py
First version of the script, using a brute-force approach

##### lola.py
Second version of the script, using a tree-like data structure for partitioning the 2d space

## input
Contains the 5 input files

## output
Contains the 5 output files

## lib
Contains the python libs

## test
Contains the unit tests
